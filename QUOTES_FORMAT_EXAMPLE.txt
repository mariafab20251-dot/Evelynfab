TIPS:
===============================================================================

1. Ensure same number of lines in both files
   - If counts don't match, system uses minimum count
   - You'll see a warning message in processing log

2. Keep subtitles visual-friendly
   - Short enough to read quickly
   - Punchy and impactful
   - Include emojis when appropriate

3. Make voiceover engaging
   - Use storytelling techniques
   - Vary your pacing and rhythm
   - Include pauses for emphasis (use punctuation)
   - End with a call-to-action

4. Test different combinations
   - Same quote, different explanations
   - Try multiple voiceover variations
   - See what resonates with your audience
